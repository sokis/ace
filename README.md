## 开发流程
项目基于`vuex-cli`构建。数据流管理用vuex2.0，数据源基于Cnode社区的API。
开发中需要注意的几个点，这里逐一罗列下：

**创建公用组件** 
```
$ vuex g component <CompName>
```

**创建路由**    
```
$ vuex g route <RouteName>
```
